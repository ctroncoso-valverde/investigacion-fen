3 Generar y desplegar

Pasos finales:
1. Descarga los dos archivos cuando aparezcan
2. En Terminal: cd ~/Documents/Repositorio/Investigacion-FEN && ./update_dashboard.command
3. El script hace backup → reemplaza → git push
4. En 1–2 minutos GitHub Pages publica los cambios.
